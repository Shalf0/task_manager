Task Manager


Перед запуском приложения необходимо:
1. Postgres с базой данных TaskManager
2. Установить зависимости из файла requirements.txt
3. Установить необходимые переменные окружения:
    - DATABASE_URL = 'postgresql://USERNAME:PASSWORD@localhost/DBNAME'
    - APP_SETTINGS = 'config.DevelopmentConfig' or 'config.ProductionConfig'
